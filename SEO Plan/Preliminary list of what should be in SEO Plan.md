# Complete SEO Components & Free Tools Checklist

## Purpose
This comprehensive checklist contains all SEO components, strategies, and free tools needed to create and execute a complete SEO plan for any website. Use this as a reference guide for building SEO strategies and creating AI prompts for SEO implementation.

---

## CORE SEO ELEMENTS (What You Need to Optimize)

### On-Page SEO Elements
- Title tags (60 characters max)
- Meta descriptions (155-160 characters)
- Header tags (H1, H2, H3 hierarchy)
- URL structure (clean, keyword-rich)
- Internal linking
- Image optimization (alt text, file names, compression)
- Content quality and depth
- Keyword placement and density
- Page speed optimization
- Mobile responsiveness
- Core Web Vitals (LCP, FID, CLS)
- Schema markup/structured data
- Breadcrumb navigation
- Canonical tags

### Off-Page SEO Elements
- Backlinks (quality over quantity)
- Domain authority
- Local citations (NAP consistency)
- Social signals
- Brand mentions
- Guest posting
- Digital PR
- Review management

### Technical SEO Elements
- XML sitemaps
- Robots.txt file
- SSL certificate (HTTPS)
- 301/302 redirects
- 404 error management
- Site architecture
- Crawlability
- Indexability
- Duplicate content issues
- Page load speed
- Server response time
- Mobile-first indexing
- Structured data implementation
- Hreflang tags (if multi-language)

### Local SEO Elements (Critical for Local Businesses)
- Google Business Profile optimization
- NAP consistency across all platforms
- Local citations in directories
- Local content creation
- Customer reviews and ratings
- Local backlinks
- Geo-targeted landing pages
- Local schema markup
- Google Maps optimization
- Location-specific keywords

### Content Strategy Elements
- Keyword research
- Search intent analysis
- Content calendars
- Blog posts (1,500+ words for competitive topics)
- Service pages
- Location pages
- FAQ pages
- Case studies/testimonials
- Video content
- Infographics
- Pillar pages and topic clusters

### Link Building Sources & Strategies
- Industry directories
- Local business directories
- Chamber of Commerce listings
- Real estate directories (Zillow, Realtor.com)
- Home improvement directories (Houzz, HomeAdvisor)
- Guest posting
- HARO (Help A Reporter Out)
- Broken link building
- Resource page links
- Competitor backlink analysis
- Skyscraper technique

## SEO IMPLEMENTATION CHECKLIST

### Phase 1: Foundation (Week 1-2)
- [ ] Set up Google Search Console
- [ ] Set up Google Analytics 4
- [ ] Set up Google Business Profile (for local businesses)
- [ ] Verify website has SSL certificate (HTTPS)
- [ ] Create/verify robots.txt file
- [ ] Create/submit XML sitemap
- [ ] Install WordPress SEO plugin (Rank Math or Yoast)

### Phase 2: Technical Audit (Week 2-3)
- [ ] Run Screaming Frog crawl (up to 500 URLs)
- [ ] Check for broken links (404 errors)
- [ ] Identify duplicate content issues
- [ ] Check mobile-friendliness
- [ ] Test page speed (PageSpeed Insights)
- [ ] Review Core Web Vitals
- [ ] Check for proper canonical tags
- [ ] Verify proper header tag hierarchy

### Phase 3: Keyword Research (Week 3-4)
- [ ] Identify primary keywords for each service/product
- [ ] Find long-tail keyword opportunities
- [ ] Analyze search intent for target keywords
- [ ] Research competitor keywords
- [ ] Identify local keywords (city + service)
- [ ] Create keyword map for website pages

### Phase 4: On-Page Optimization (Week 4-6)
- [ ] Optimize title tags for all pages
- [ ] Write compelling meta descriptions
- [ ] Optimize header tags (H1, H2, H3)
- [ ] Optimize images (alt text, file names, compression)
- [ ] Add schema markup to pages
- [ ] Improve internal linking structure
- [ ] Optimize URL structure
- [ ] Create/optimize content for target keywords

### Phase 5: Local SEO (Week 6-8)
- [ ] Optimize Google Business Profile completely
- [ ] Build citations on top 20 directories
- [ ] Ensure NAP consistency across all platforms
- [ ] Add local schema markup
- [ ] Create location-specific landing pages
- [ ] Encourage and respond to customer reviews
- [ ] Build local backlinks

### Phase 6: Content Strategy (Ongoing)
- [ ] Create content calendar
- [ ] Write blog posts targeting keywords
- [ ] Create FAQ pages
- [ ] Develop case studies/testimonials
- [ ] Create pillar content
- [ ] Optimize existing content

### Phase 7: Link Building (Ongoing)
- [ ] Submit to relevant industry directories
- [ ] Pursue HARO opportunities
- [ ] Reach out for guest posting
- [ ] Identify broken link opportunities
- [ ] Analyze competitor backlinks
- [ ] Build relationships for natural links

### Phase 8: Monitoring & Reporting (Ongoing)
- [ ] Monitor rankings in Google Search Console
- [ ] Track traffic in Google Analytics
- [ ] Monitor backlinks
- [ ] Check for technical errors monthly
- [ ] Track local rankings
- [ ] Review and respond to reviews
- [ ] Create monthly performance reports

## PROMPT CREATION TEMPLATE

When creating AI prompts for SEO tasks, use this structure:

```
Context: [Business type, location, target audience]
Goal: [Specific SEO objective]
Requirements: [Use information from this checklist]
Constraints: [Budget = $0, free tools only]
Output Format: [What you need delivered]

Example:
"I need to create a comprehensive SEO plan for a window treatment business (Shaded In The Sun) serving Edmond and OKC area. Using only free tools from the SEO checklist, create a 90-day action plan that covers: technical SEO audit, local SEO optimization, keyword research for 'window treatments + location' terms, and content strategy. Include specific free tools to use for each task and expected timelines."
```
