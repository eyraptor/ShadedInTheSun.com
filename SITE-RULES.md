# Site Rules

## Meta Description Rules (Updated July 2026 — replaces the old 135-char cap)
- Hard cap: 155 characters. Nothing over 155, ever.
- Front-load rule: the core value proposition + a city name (Oklahoma City, OKC, or Edmond) must appear within the first 105 characters. Characters 106–155 are bonus copy for desktop (CTA, differentiator, "Free in-home consultation").
- No phone numbers in any meta description.
- Brand names (Graber, Norman) allowed ONLY on brand-specific pages: graber.html, norman.html, graber-ultralite-lift-system.html, norman-smartdrape-oklahoma-city.html, why-is-hunter-douglas-so-expensive.html. No brand names in metas on any other page.
- `og:description` must exactly match the meta description on every page. If you change one, change both.
- `og:title` must match the page `<title>`.

## Content Rules (unchanged — do not violate)
- Existing shutters pages and content stay exactly as they are. Do not add NEW shutters content anywhere.
- Approved brands in content: Graber and Norman® only. Never mention Draper.
- Do not change any URLs or file names (static Porkbun hosting — renames break everything).
- Do not invent product claims. Only use facts already present on the site.
